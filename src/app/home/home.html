LisTo-Do La mejor forma de lograr algo, es haciendo algo para lograrlo.

Esta aplicación le ayudara a manejar sus tareas personales de forma clara y sencilla para lograr sus objetivos.
Para poder usar la aplicación es necesario crear una cuenta de usuario con su correo, una vez creada podra agregar y manejar sus tareas personales iniciando sesión con su correo y contraseña.
